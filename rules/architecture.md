# アーキテクチャ

## ディレクトリ構成

```
src/
├── app/
├── components/
├── features/
├── lib/
├── utils/
└── types/
```

### app

ルーティングを管理するディレクトリです。
apiルートもここで管理します。

### components

ライブラリからインストールして使用する場合はこちらに配置します。

### features

appのルーティングされるディレクトリと同じになるようにします。

### lib

ライブラリを管理するディレクトリです。

### utils

ユーティリティを管理するディレクトリです。

### types

型定義を管理するディレクトリです。

## フォームライブラリを使用について

今回はReactHookFormを使用します。
要件に上がっているのはフォーム1つだけですが、今後フォームが増えても管理しやすいようにフォームライブラリを使用します。

## 状態管理ライブラリ

一旦は使用せず、`useState`や`Provider`を使用します。
現状の要件はグローバルStateを用意してなくても実現可能なので必要になったタイミングで導入する。
